## Data Scraping Projects (Python)

Python project demonstrating web scraping and API data collection.

### Overview
Built scripts to extract data from websites and APIs, convert the data into structured datasets, and export them for further analysis.

### Workflow & Tools
- Used **Requests** to retrieve website and API data  
- Parsed HTML using **BeautifulSoup**  
- Extracted product, quote, and book information from multiple pages  
- Converted raw data into structured **Pandas DataFrames**  
- Exported datasets to **CSV files**

### Deliverables
- Books dataset (`books.csv`)
- Quotes dataset (`quotes.csv`)
- Laptop products dataset (`laptops.csv`)
- Fake Store API dataset (`fakestore.csv`)

### Skills
Python, Web Scraping, BeautifulSoup, APIs, Data Collection, Pandas, Data Extraction
